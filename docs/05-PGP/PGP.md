# PGP 

1) Introduccion
a. Proposito y alcance 

El objetivo de este proyecto es facilitar y agilizar la gestion de turnos, reservas, asistencias y metricas del centro de actividades deportivas.
El alcance abarca a todos los miembros que se relacion de forma directa con el sistema. Esto engloba a los empleados, clientes y administradores del centro.

b. Definiciones, acrónimos y abreviaturas a considerar

ABM: Alta, baja y modificacion
RF: Requerimiento Funcinal
HU: Historias de usuario
RNF: Requerimiento no funcional
